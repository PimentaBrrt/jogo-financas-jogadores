# Jogo de Finanças: Site dos Jogadores

Esse repositório contém uma parte do jogo. O site criado aqui deve ser aberto em uma das máquinas para jogar. Ele possui as cartas na mão de cada um dos jogadores, e possui um botão de "Passar a vez" para mostrar a mão do próximo jogador.
